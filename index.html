Overview

This is the R Interface to Open Neural Network Exchange (ONNX) - a standard format for models built using different frameworks (e.g.TensorFlow, MXNet, PyTorch, CNTK, etc). It defines an extensible computation graph model, as well as definitions of built-in operators and standard data types. Models trained in one framework can be easily transferred to another framework for inference. This open source format enables the interoperability between different frameworks and streamlining the path from research to production will increase the speed of innovation in the AI community.

Installation

  • First, follow instructions here to install the onnx Python package.
  • Then install the development version of the R package on Github via the following:
if (!require(devtools)) install.packages("devtools")
devtools::install_github("onnx/onnx-r")
